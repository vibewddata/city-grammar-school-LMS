# 🎓 City Grammar School LMS

A complete **Full-Stack School Management System** for **City Grammar School Bahawalnagar** built with modern technologies.

## ✨ Features
- **Role-based Authentication** (Admin, Teacher, Student, Accountant)
- **Admin Dashboard**: Manage students, teachers, classes, fees, salaries
- **Student Portal**: Fee challans, attendance, results, profile
- **Teacher Portal**: Mark attendance, upload results, salary slips
- **Fee & Salary Management**: PDF generation, payment tracking
- **Reports & Analytics**: Charts, exports, filters
- **Responsive Design**: Mobile-first, Dark mode
- **Modern UI**: Tailwind CSS, React, Chart.js
